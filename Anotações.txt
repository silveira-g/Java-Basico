Todo o código da classe fica dentro das chaves { }; existem exceções que são os métodos.
Se a classe for executável, precisa ter um método especial denominado MAIN
Metodo main segue um padrão: public static void main (String[] args) {}

Padrão de escrita:
O nome da classe tem que estar de acordo com o nome do arquivo.
Todos os arquivos a serem criados as primeiras letras sempre em MAIUSCULO.
Toda variável tem que ser escrita em letra minuscula, mas se for composta a primeira letra da próxima palavra é maiuscula. Esta prática se chama camelCase Ex: minhaVariavel
Caso a variável seja constante, é criado com a denominação final e o nome da variável toda MAIUSCULA. Ex final String BR = 'Brasil";

Identação:
Escrever o código em forma de hierárquica para facilitar visualização e entendimento do programa.

Tipos de variaveis:
Primitivos:
int, byte, short, long, float, double, boolean e char

Operadores:
Operador de atribuição: é representado pelo símbolo de igual (=)
Serve para atribuir um valor a uma variável.

Operadores aritméticos: adição (+) subtração (-) multiplicação (*) divisão (/) modulo (%)
o operador de adição quando utilizado com variáveis strings irá realizar a concatenação de textos.

Operador unários: operador de deixar numero positivo (+), de negar um numero (-), incrementar numero (++), decrementar numero (--) ou negar uma expressão booleana(!)

Operador ternário: forma resumida para definir uma condição (?)
Ex: a==b ? c : d

Ooperador relacional: >, <, >=, <=, == etc

Operadores lógicos: && (e) || (ou)


Métodos:
Correspondem a funçoes ou sub-rotinas dentro de classes.

Declaração de métodos:
TipoRetorno NomeObjetivoNoInfinitivo Parametro(s)
Ex:
int somar (int numeroUm, int numeroDois)
Caso o método não retorne valor ele será representado por void.

Escopo: o ambiente uma variável pode ser acessada.

Palavras reservadas:
Identificadores da linguagem que já possuem finalidade portanto não podem ser usadas para criar variáveis metodos ou classes.

Scanner:
Classe que permite interação mais assertiva do usuario.
